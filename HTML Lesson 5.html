HTML Fundamentals Lesson

Introduction to HTML

  • HTML stands for HyperText Markup Language, used as the standard markup language for creating Web pages.
  • It describes the logical layout and structure of a Web page using a series of elements (tags); instructs the browser on how to display content.
  • HTML code components completely form the core foundational building blocks of all modern websites, allowing browsers to display text and images.

Basic Structure

  • Every single HTML document must strictly start with a precise document type declaration: <!DOCTYPE html>; defined by a start tag, element content, and an end tag.
  • The base of the document begins identically with <html> and formally terminates with </html>, serving as the required root element.
  • The directly visible content logic of the document is safely positioned directly between the <body> and </body> functional tags.

Common Elements

  • Major headings are defined strictly with the <h1> to <h6> tags, representing levels of importance.
  • Standard text paragraphs are constructed dynamically using the <p> tag and are usually used to structure typical text blocks securely.
  • The document's body content is represented by the <body> tag.
  • Images can be embedded using the self-closing <img> tag without any strict closing tag being needed.

Attributes and Links

  • Functional attributes provide necessary additional information about HTML elements.
  • The attributes are always identified and specified within the start tag.
  • For example, the absolute href routing attribute is used to specify the URL of the page the link goes to.

Forms and Input

  • An HTML form is used to collect user input that is often sent to a server for processing.
  • A standard <button type="submit"> component is necessary to submit the form.
  • Another example is the <input> element which is versatile, changing its interactive interface behavior based on the associated type attribute (text/password).
