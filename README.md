# Crack The Code

An interactive web-based application to teach the fundamental concepts of programming languages: JavaScript, CSS, and HTML.

## Pages Overview

| Page | File | Description |
|------|------|-------------|
| Homepage | `Homepage 5.html` | Landing page introducing the platform with options to sign up or log in. |
| Signup | `Signup 5.html` | Account creation page with a registration form. |
| Login | `Login 5.html` | User login page to access the dashboard. |
| Dashboard | `Dashboard 5.html` | Main hub with links to all available lessons and a logout option. |
| HTML Lesson | `HTML Lesson 5.html` | Lesson covering the fundamentals of HTML. |
| CSS Lesson | `CSS Lesson 5.html` | Lesson covering the fundamentals of CSS. |
| Java Lesson | `Java Lesson 5.html` | Lesson covering the fundamentals of Java. |
| Extra Resources | `Extra Resources 5.html` | Additional learning resources and references. |

## Navigation Flow

```
Homepage --> Signup --> Dashboard
Homepage --> Login  --> Dashboard

Dashboard --> HTML Lesson --> CSS Lesson --> Java Lesson --> Extra Resources
```

Each lesson page has navigation buttons to move to the next or previous lesson, and a button to return to the Dashboard.

## How to Run Locally

1. Download or clone this repository to your computer.
2. Open the project folder in your file explorer.
3. Double-click on `Homepage 5.html` to open it in your default web browser.
4. Use the on-screen buttons to navigate through the site.

No web server, installations, or dependencies are required. All pages are static HTML files that run directly in any modern web browser (Chrome, Firefox, Edge, etc.).
