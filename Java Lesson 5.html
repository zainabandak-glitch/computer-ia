Introduction to Java

  • Java is a high-level, object-oriented programming language developed by Sun Microsystems in 1995. It is mostly used for building desktop applications, web applications, Android apps, and enterprise systems.
  • The main features of Java include:
    • Object-Oriented Programming: Java supports Object-Oriented Programming concepts to create modular and reusable code.
    • Platform Independence: Java programs can run on any operating system with a Java Virtual Machine.
    • Security: Java ensures reliability and security through strong memory management and exception handling.
    • Multithreading and Concurrency: Java allows concurrent execution of multiple tasks for efficiency.
    • API and Standard Libraries: Java provides extensive built-in libraries for various programming needs.
    • Frameworks for Enterprise and Web Development: Java supports frameworks that simplify enterprise and web application development.
    • Open-Source Libraries: Java has a wide range of libraries to extend functionality and speed up development.
    • Maintainability and Scalability: Java's structured design allows easy maintenance and growth of applications.

Basic Syntax

  • In Java Programming, the following syntax are important:
    • Case Sensitivity: Java is case sensitive, which means the identifiers Hello and hello would have different meanings in Java.
    • Class Names: For all class names, the first letter should be in upper case. If several words are used to form a name of the class, each inner word's first letter should be in upper case.
    • Method Names: All method names should start with a lower case letter. If several words are used to form the name of the method, then each inner word's first letter should be in upper case.
    • Program File Name: The name of the program file should exactly match the class name. When saving the file, you should save it using the class name and append '.java' to the end of the name - if the file name and the class name do not match, your program will not compile. Also, in case you do not have a public class present in the file, then the file name can be different than the class name. For example, assume 'MyFirstJavaProgram' is the class name. Then the file should be saved as 'MyFirstJavaProgram.java'
    • public static void main(String args[]): Java program processing starts from the main() method, which is a mandatory part of every Java program.

Data and Variables

  • Variables are containers for storing data values, and there are different types of variables, for example:
    • String - stores text, such as "Hello". String values are surrounded by double quotes
    • int - stores integers (whole numbers), without decimals, such as 123 or -123
    • float - stores floating point numbers, with decimals, such as 19.99 or -19.99
    • char - stores single characters, such as 'a' or 'B'. Char values are surrounded by single quotes
  • Variables hold data permanently in memory, and are formally bounded by primitive data types like int for integers and boolean for true/false.

Control Flow

  • Control flow logic conditionally dictates and determines the dynamic order in which statements are evaluated and executed in a program.
  • Conditional decision-making statements like if-then and switch logically execute entirely different code paths based on values.
  • Iterative looping algorithms like for and while repeatedly construct block executions sequentially until an exit condition triggers.
